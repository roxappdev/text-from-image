Tesseract.js OCR Демо

Демонстрация работы библиотеки Tesseract.js для распознавания текста с изображений прямо в браузере.

Запустить OCR Демо

Возможности демо:

  • Распознавание текста с изображений в форматах PNG, JPG, JPEG, BMP, GIF
  • Поддержка русского и английского языков
  • Предпросмотр загруженного изображения
  • Отображение прогресса распознавания
  • Копирование распознанного текста в буфер обмена
  • Адаптивный интерфейс

Технологии: Tesseract.js, HTML5, CSS3, JavaScript

Для работы требуется запущенный сервер разработки (npm start)
